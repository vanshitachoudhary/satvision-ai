# 🚀 Overview

CloudFree Bharat is an AI-assisted satellite intelligence platform that detects cloud-covered regions in satellite imagery, reconstructs obscured areas using computer vision techniques, and generates land intelligence analytics such as vegetation cover, water detection, and land assessment.

The project aims to improve the usability of satellite imagery for agriculture, disaster monitoring, environmental analysis, and remote sensing applications.

---

# ❗ Problem Statement

Satellite imagery plays a vital role in monitoring Earth. However, cloud cover often hides important land features, making accurate analysis difficult.

This impacts:

- 🌾 Precision Agriculture
- 🚨 Disaster Management
- 🌍 Environmental Monitoring
- 💧 Water Resource Analysis
- 🛰️ Remote Sensing

CloudFree Bharat addresses this challenge by enhancing satellite images and extracting meaningful land intelligence.

---

# 💡 Solution

CloudFree Bharat performs the following tasks:

- ☁️ Detects cloud-covered regions
- 🎯 Generates accurate cloud masks
- 🧠 Removes cloud-obscured regions
- 🌿 Analyzes vegetation
- 💧 Detects water bodies
- 📊 Generates land intelligence
- 📷 Produces enhanced satellite imagery

# 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/vanshitachoudhary/satvision-ai.git
```

### Move into Project

```bash
cd satvision-ai
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

or

```bash
streamlit run app.py
```
